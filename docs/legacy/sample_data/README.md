# docs/legacy/sample_data/

This directory is **gitignored** — files here are never committed.

It exists as a local working area for files that were previously at the repository root
during the Python project era. They contain real user credentials and personal data,
so they must not be committed.

## Files that belong here (local dev only)

| File | Sensitivity | Description |
|---|---|---|
| `info.db` | HIGH — real GPS + VIN | SQLite telemetry database (position, battery, charge, SOH history) |
| `info.db-shm` | HIGH | SQLite shared-memory file (transient, paired with info.db) |
| `info.db-wal` | HIGH | SQLite write-ahead log (transient, paired with info.db) |
| `config.json` | **CRITICAL** | Stellantis OAuth refresh tokens + client secrets |
| `cars.json` | HIGH — real VIN | Vehicle metadata cache with VINs and vehicle UUIDs |
| `otp.bin` | **CRITICAL** | Pickled MQTT OTP session — grants remote-command capability |
| `charge_config.json` | LOW | Per-VIN charge control thresholds and stop hours |
| `activity.log` | MEDIUM | Application log (may contain VINs, charge events) |
| `mypeugeot.apk` | MEDIUM | Official MyPeugeot APK — source for client_id / client_secret extraction |
| `mypeugeot.apk.bz2` | MEDIUM | Compressed APK — same as above |

## How to recreate for local development

1. **info.db** — run the legacy Python app with `-r` flag (recording mode), or use the
   Phase 3 import tool (`tools/import_legacy_db/`) to migrate an existing database.

2. **config.json** / **otp.bin** — run `InitialSetup` from
   `psa_car_controller/psa/setup/app_decoder.py` with a valid brand APK, email, and password,
   then complete OTP setup via SMS.

3. **cars.json** — auto-generated by the Python client on first `/vehicles` call.

4. **mypeugeot.apk** — download from the Google Play Store or use the GitHub release link
   referenced in `app_decoder.py`.

## After Phase 2

Once the Dart port is complete, `info.db` can be imported into Isar using
`tools/import_legacy_db/`. The `config.json` and `otp.bin` are superseded by the
Flutter app's Keychain/EncryptedSharedPreferences storage.
