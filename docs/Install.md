# Installation

1. Download or clone the app.

2. Install requirements:

- Python >= 3.11 recommended

```bash
pip install psa-car-controller
```

3. Start the app (charge control enabled):

```bash
psa-car-controller --web-conf
```

4. Open:

```text
http://localhost:5000
```

5. Complete setup from the PWA `Setup` tab:

- Login setup (brand/email/password/country)
- OAuth code completion
- OTP SMS + PIN finalization

6. Optional: install the PWA from your browser menu (desktop/mobile).

Use `psa-car-controller -h` to list all CLI options.
